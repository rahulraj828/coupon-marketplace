## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/rahulraj828/coupon-marketplace.git
   ```

2. Install dependencies:
   ```bash
   cd coupon-marketplace
   npm install
   ```

3. Start the server:
   ```bash
   npm start
   ```

4. Open your browser and visit:
   ```
   http://localhost:3000
   ```
